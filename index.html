CSS syntax ประกอบด้วยอะไรบ้าง
ประกอบไปด้วย : Selector (e.g. class,id), property (e.g. color, font-size), value (e.g. "red", 26)
การเขียน CSS มีกี่แบบอะไรบ้าง
การเขียน CSS มี 3 แบบ : Internal (เขียนใน .html), External (เขียนไฟล์แยกแล้วใส่ link), Inline (เขียนใน tag)
selector มีกี่แบบอะไรบ้าง
selector มี : Class (.), ID (#), Tag (div,p), Pseudo (::before, ::after), แบบ combinator (>,+,~)
combinator มีกี่แบบอะไรบ้าง
combinator มี 4 แบบ: (space) ตัวที่อยู่ข้างใน, child (>) ตัวลูก, (+) เลือกตัวถัดไป , (~) general -> อันที่มีลักษณะตามที่ระบุ
property และ value คืออะไรแตกต่างกันอย่างไร
property คือค่าที่ให้เลือกว่าจะปรับอะไร e.g. color,background-color,height. Value คือค่าที่กำหนดให้ property e.g. red, 12px, bold
property ที่สอนใน precamp มีกี่กลุ่มอะไรบ้าง
Display:flex, flex-direction (row) Default

font-size(20px), color(blue),font-style (italic)

width: 80px;height:20px;background-color: red;

font-size (20px)

font-weight (bold)

value unit มีกี่ประเภทอะไรบ้างแต่ละ ประเภทคืออะไร
มี 2 ประเภท : Relative กับ fixed
  • Relative จะปรับตาม container parent (e.g. 100% คือ 100% ของ parent container))
  • Fixed จะไม่สนใจ parent element ค่าจะ fixed (e.g. 12px)
